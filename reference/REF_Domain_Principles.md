# Domain Principles

Hard-won lessons from prior analysis. Each entry exists because a mistake was made. Read before making recommendations.

---

## Card text discipline

- **Read the card before recommending it.** Pattern-matching from card name or art has been wrong repeatedly. When a card is new, unfamiliar, or from a UB set, look it up on Scryfall before endorsing a swap.
- **Sephiroth, Fabled SOLDIER is a sac outlet and payoff, not a token producer.** Do not count him toward token-doubling synergies (e.g., Anointed Procession).
- **Master of Etherium's characteristic-defining ability applies in all zones, including the graveyard.** Relevant for reanimator, recursion, and graveyard interactions.
- **Venser, Shaper Savant is legendary.** Tokens of Venser die to the legend rule. Relevant when a clone strategy considers him as a copy target.
- **Aggravated Assault cannot be activated with firebending mana.** It is a sorcery-speed activation restricted to the main phase; firebending mana expires at end of combat. For a Zuko / firebending deck, use **Hellkite Charger** for infinite combat.
- **Zuko Exiled Prince and Fire Lord Zuko cannot coexist.** Same legendary name root; legend rule applies.

## Evaluation discipline

- **Mechanical distinctiveness is a hard filter.** Two decks that play the same way are one deck too many, regardless of flavor or individual card quality.
- **Honest scoring over optimistic scoring.** Inflated scores will be caught. Score the deck as it actually plays, not as it flatters the builder.
- **Power-first, collection-constrained second.** Build the strongest legal list first, then resolve against collection/budget.
- **Count token producers before recommending token doublers.** A doubler with only 2–3 token generators is a dead card most games. For Teysa aristocrats: Desecrated Tomb, Woe Strider, Endrek Sahr are the real producers. Sephiroth is *not*.
- **No single axis compensates for a catastrophic gap in another.** A deck at 5/5/5/1 will lose games it should win.

## Workflow discipline

- **Game Changer list is authoritative. Memory is not.** Always cross-reference `REF_Game_Changers_List.md` when evaluating GC count. The list changes.
- **Surplus check before replacement search.** When a card appears missing, check the Full Card Matrix for surplus *first*. Cards are often physically lost from a surplus copy rather than genuinely unowned.
- **Reskin alias check before declaring unowned.** See `REF_Reskin_Aliases.md`.
- **File ground truth.** When a stored summary and the `.txt` decklist conflict, the `.txt` wins.
- **Surgical edits over rewrites.** When fixing a document, change only what needs changing. Don't restructure silently.
- **Verify card text before recommending.** See "Card text discipline" above. Non-negotiable for UB sets.

## Known GC list specifics

Ground truth is `REF_Game_Changers_List.md`. These are common points of confusion:

- **Deflecting Swat: OFF the list** (removed October 2025).
- **Doubling Season: NOT a Game Changer.** Frequently assumed to be; it isn't.
- **On the list and commonly relevant:** Smothering Tithe, Gamble, Jeska's Will, Teferi's Protection (→ Aang's Shelter), Enlightened Tutor, Mana Drain, Cyclonic Rift, Drannith Magistrate, Fierce Guardianship, Rhystic Study, Mystic Remora.

Always re-verify against the current list before finalizing a deck.

## Collection realities

- **Kelz0r.dk, Faraos Cigarer, MtgWebshop.dk** are the primary Danish retailers. Cardmarket.eu is the main cross-border singles source.
- **Avoid UK/US stores** — PostNord handling fees make small orders uneconomic.
- **Proxies are allowed** for testing and cross-deck staples. Physical acquisition is preferred for decks that see regular play.
